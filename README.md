#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
https://github.com/bensuperpc/docker-whitespace.git
```
```sh
cd docker-whitespace
```
#### Usage

```sh
./compiler.sh ws2c <file>
```
#### Build
```sh
make linux/amd64
```
